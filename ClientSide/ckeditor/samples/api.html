Allin Qillqay » Corrector Ortográfico Online para el Quechua Cusqueño

Esta es una DEMO del Corrector Ortográfico para el Quechua Cusqueño implementado con Foma por Annette Rios. Esta DEMO hace uso de SCAYT, Dojo para enviar las consultas y recibir los resultados del algoritmo de búsqueda MED "Distancia Mínima de Edición". También se hace uso del CKEditor, una interface web similar a la de un Procesador de Textos. Para mayor información sobre la implementación de este Servicio Web escribir a Richard A. Castro Mamani con el siguiente correo rcastro@hinant.in.
